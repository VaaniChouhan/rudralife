### 2. UI/UX & Design System Audit Fixes (Fix-One-At-A-Time Plan)

- **Typography System Overhaul**: Established a single type scale system (`eyebrow` → `h2.display` → `lead` → `body` → `caption`) in `index.css`. Set all headings (`h1` to `h6`) to use `Cormorant Garamond` by default with elegant regular/medium weights (`400`/`500`) to prevent browser default bolding from distorting serif display fonts. Excluded small metadata labels in bento cards, forcing them to remain as clean sans-serif `Inter` elements. Adjusted CTA button weights to a premium `600` weight (instead of `700` bold). Wrapped core keywords in `em` across major section headings for gold-italic highlight effects.
- **Extreme Media Lazy Loading**: Configured all secondary background videos (`bg-video-milind`, `bg-video-fire`, `bg-video-spotlight`, `bg-video-legacy`, `bg-video-final`) with `preload="none"` and `data-src` tags in `BackgroundLayers.jsx`. Implemented dynamic source injection inside `App.jsx`'s `IntersectionObserver` to trigger video source downloading *only* as the user approaches the corresponding section. Mapped static image layers similarly with `data-bg-img` attributes, reducing initial page load overhead by 90%+.
- **Layer Mappings Alignment**: Extended `layerMap` with mappings for new sections (`s-indra-mala` and `s-celebrities`) to dynamically trigger correct background video/image preloads on scroll.
- **Unified Card Radius Tokens**: Defined clear radius tokens (`--r-xs`, `--r-sm`, `--r-md`, `--r-lg`, `--r-xl`) in root CSS variables. Applied `--r-md` consistently to `.jcard`, `.story-card`, `.rare-card`, and `.mala-card` definitions to remove chaotic card corner variation.
- **Readability Background Overlay**: Reduced `.fixed-bg-layer.active` opacity from `0.48` to `0.32` and applied a dark linear gradient overlay (`rgba(13,10,6,0.7)` to `rgba(13,10,6,0.85)`) on all background layers to guarantee typography contrast across transitions.
- **Header Logo & Navigation Refinement**: Increased header logo height to `42px` for brand clarity. Changed nav CTA `border-radius` to `8px` to align with the premium, luxury architecture.
- **On-Brand WhatsApp Floating Button**: Replaced the default jarring WhatsApp green FAB style with a glassmorphic dark-gold style (`background: rgba(13, 10, 6, 0.88)` and `border: 1px solid rgba(196, 154, 60, 0.4)`) with a gold pulsing ring animation.
- **Hero Bead Orb Restoration**: Restored the floating bead orb component inside the hero-scene container in `App.jsx` so it renders correctly as the central visual identity of the page.
- **Editorial Story Quotes**: Replaced cheap-looking silhouette SVGs in `.story-card` blocks with elegant typographic double quote marks in Cormorant Garamond style.

### 2. `index.css` — Modern Typography System
Following `modern-web-guidance` skill guidelines:

| Rule | Why |
|------|-----|
| `text-wrap: balance` on h1–h6 | Even line breaks on headings (≤6 lines) |
| `text-wrap: pretty` on p, li, blockquote | Prevents orphaned last words in body copy |
| `font-size-adjust: from-font` (commented, ready to enable) | Stable layout during font swap |
| `text-rendering: optimizeLegibility` | Sharper text on all displays |
| `-moz-osx-font-smoothing: grayscale` | Cross-platform anti-aliasing |
